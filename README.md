# Session & Cookie Authentication Demo
A Node.js/Express application demonstrating form validation, session management, and cookie handling.

## Features
- Client-side password validation (length, case, numbers, special chars) using ASCII
- Email and phone number format validation
- Session-based user data storage
- Cookie management
- Logout functionality with session destruction

## Setup
npm install
node server.js
# Visit http://localhost:4040

## Dependencies
- express
- cookie-parser
- express-session
